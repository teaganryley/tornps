# Tornps

![The Stalk Market](./docs/images/tornps.jpeg)

Library of functions used for analyzing/predicting Sow Joan's Stalk Market.

## Installation:

1. Clone repo
2. Create Conda environment using the spec-file: `conda create --name myenv --file spec-file.txt`
3. Navigate to project root and `conda activate myenv`
4. Install from the project root: `pip install -e .`
